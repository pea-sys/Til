## Web ページで使用されている技術を検出する

Chrome 拡張の Wappalyzer - Technology profiler を使用することで  
Web ページで使用されている技術を検出できる。

https://chrome.google.com/webstore/detail/wappalyzer-technology-pro/gppongmhjkpfnbhagpmjfkannfbllamg?hl=ja
